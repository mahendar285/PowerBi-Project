# IPL Analysis Dashboard

This is a Power BI dashboard project created to analyze IPL data from 2008 to 2025. The goal of this project was to convert raw cricket data into meaningful insights using Power BI, SQL, and Excel.

While building this dashboard, I worked on data cleaning, data modeling, creating relationships between tables, writing DAX measures, and designing interactive visualizations.

## What the Dashboard Shows

* Team-wise performance analysis
* Home vs Away match comparison
* Top batting performers
* Season-wise IPL statistics
* Captain analysis
* Interactive filters for teams and seasons
* Key metrics such as total matches, total wins, win percentage, and total runs
